Aero Tuning

If the vehicle is tuned properly it should NOT require aero at all.
Drive the vehicle around first and see if it will require it, then add it if needed.
If aero IS added, Set front and rear to the lowest setting to start with.

Guidelines
  • Understeering: Add a small amount to the front
  • Rear Sliding Out In Corners: Add a small amount to the rear
Tuning Logic
  • Front Downforce: More turning grip, Less top speed
  • Rear Downforce: More rear corner grip, Less top speed
  • Both Downforce: May help grip, Way less top speed
Pro-Tip

DO NOT use Aero at all!, IF FORCED use very little!
